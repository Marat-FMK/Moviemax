## 📱 О приложении
Мобильное приложение в котором вы найдете более миллиона фильмов и сериалов. Приложение позволет пользователям находить интересные фильмы и получать персональные рекомендации. 
На главной странице — карусель с популярными и рекомендованными фильмами. 
Реализован удобный поиск по названию, а также фильтры по рейтингу и жанрам.
По названию фильма можно найти похожие фильмы с полным описанием картины, информацией о съемочной группе.
В разработке добавить поиск сериалов 


## 🛠️ Технологии
Swift 5, IOS 16.0+, UIKit, SwiftUI, MVVM, AppStorage / UserDefaults, CoreData, URLSession, API, CoreData, Firebase
